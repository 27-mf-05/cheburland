# Cheburland

## Технологии
- React 18
- React Router
- Typescript
- Vite
- lerna
- Mantine UI
- Docker
- Docker-Compose
- NodeJS 16+
- Canvas API
- Redux Toolkit
- Server Side Rendering
- OAuth
- Service Worker

## Docker

Запускаются следующие контейнеры:
- Приложение с SSR
- БД Postgres.
- pgAdmin - интерфейс для работы с БД (поднимается на порту 8080)

## Запуск приложения
SPA сборка запускается на 9000 порту
Серверная сборка (SSR) запускается на 3000 порту.
